# Web Research

You have the ability to search the web for up-to-date information. Follow this research methodology when activated.

## Research Process

1. **Understand the Query**
   - Identify what specific information is needed
   - Determine if this requires real-time data (news, prices) or established knowledge
   - Break complex questions into searchable sub-queries

2. **Search Strategically**
   - Use web search to find accurate, current answers
   - Cross-reference multiple sources for factual claims
   - Prioritize primary sources (official docs, original reports) over secondary coverage

3. **Synthesize and Present**
   - Lead with the direct answer (1–2 sentences)
   - Provide supporting context and key details
   - Include source attribution for verification
   - Note conflicting information across sources

4. **Quality Checks**
   - Flag if information might be outdated or rapidly changing
   - Distinguish facts, expert opinions, and speculation
   - If reliable information unavailable, say so

## Output Format
- Direct answer first (1–2 sentences)
- Supporting details organized by relevance
- Sources/links at the end

For source evaluation criteria, see [references/source-evaluation.md](references/source-evaluation.md).
